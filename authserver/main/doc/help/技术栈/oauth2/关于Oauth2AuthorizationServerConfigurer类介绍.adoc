关于 Oauth2AuthorizationServerConfigurer 类介绍, 理解当前类针对实现Oauth2认证，默认配置了什么内容。

OAuth2AuthorizationService用于管理新的和现有的授权
OAuth2AuthorizationConsentService用于管理新的和现有的授权同意

OAuth2TokenGenerator用于生成 OAuth2 授权服务器支持的令牌

createConfigurers() 方法配置了一下内容:
初始化配置客户端身份验证  OAuth2ClientAuthenticationConfigurer
授权服务器元数据端点的配置器 OAuth2AuthorizationServerMetadataEndpointConfigurer
初始化认证端点  OAuth2AuthorizationEndpointConfigurer
初始化token授权  OAuth2TokenEndpointConfigurer
令牌自省端点的配置器 OAuth2TokenIntrospectionEndpointConfigurer
令牌撤销端点的配置器 OAuth2TokenRevocationEndpointConfigurer

oidc 配置中包含以下配置
providerConfigurationEndpoint OpenID Connect 1.0 提供商配置端点的配置器
userInfoEndpoint OpenID Connect 1.0 UserInfo 端点的配置器
clientRegistrationEndpoint OpenID Connect 1.0 客户端注册端点的配置器
